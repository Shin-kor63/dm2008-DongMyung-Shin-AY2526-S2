Arduino LED Control

This project explores how to power and control an LED using Arduino and a breadboard.

Through guided demos, I tried three different ways to control an LED and understand how each method works.

### What I learned

- How to connect an LED to Arduino using a breadboard  
- How to supply power to an LED  
- Basic Arduino control methods  

### Methods

1. **Constant Power
   The LED stays on all the time.

2. **Digital Output**  
   The LED turns on and off.

3. **PWM Output**  
   The LED brightness changes (dimming and fading).
